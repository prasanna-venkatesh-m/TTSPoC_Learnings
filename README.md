# TamilTTS_Playground

A small playground for experimenting with Tamil text-to-speech (TTS) and chatbots.  

- Try Tamil TTS using Python (gTTS).  
- Test LLMs for generating spoken, colloquial Tamil.  
- Proof-of-Concepts (POC) and learning experiments.  

Folder structure (example):  
- notebooks/ – Jupyter notebooks  
- scripts/ – Python scripts  
- models/ – saved models or outputs  
- docs/ – notes and references  

This repo is for learning and experimentation, not production.
